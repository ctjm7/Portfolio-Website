Case Study  -  See You at the Movies! App

The challenge

To create a web app that provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The goal

Build a project from backend to frontend to develop my skills as full-stack Software Engineer. This was part of the curriculum in CareerFoundry's Web Development Program. To produce an application that utilizes the "MERN" stack (MongoDB, Express, React, and Node.js).

The process

Server-Side:

A database was created using MongoDB, a non-relational database. The data for the movies collection is stored in JSON format. Access was provided via a RESTful API built with Node.js and Express. The API was tested using Postman. A basic HTTP and JWT (token-based) authentication were implemented. movie-api on Github

Client-Side:

This is a single-page application using React. The app consists of several components, interface views. The login gives you access to the main view, showing a list of all movies. When you can enter the single movie view it provides movie information and the ability to add to your favorites list. A profile view shows user information and the ability to update user info. A registration view is accessible if a user has not yet signed up. movie-api-client on Github

The results

This was a rewarding and at times challenging endeavor. I was introduced to several new software applications. It involved learning and familiarizing myself with React. I also had to dive into debugging and referencing documentation. This process taught me perseverance and the need to look at problems from different angles. Now after all that hard work we should all take a break. "See You at the Movies!"
